# laptop-upgrade-os-deployment-lab
Laptop hardware upgrade and Windows 10 Pro deployment lab (CompTIA A+)
Laptop Upgrade & OS Deployment Lab (CompTIA A+)
📌 Objective:

To perform a full laptop hardware upgrade and operating system reinstallation to simulate real-world IT support tasks.

🛠️ Lab Activities:
Disassembled laptop and accessed internal components
Upgraded storage from HDD to NVMe SSD
Upgraded RAM from 8GB to 32GB (3200 MHz)
Installed Windows 10 Pro (clean installation)
Verified drivers and system functionality
